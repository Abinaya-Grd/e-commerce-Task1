# Task 1: E-Commerce Store

## Project Description
This project is a basic e-commerce web application developed using Django.  
It allows users to view products, add them to a cart, and place orders.

## Features
- User registration and login
- Product listing with details
- Shopping cart functionality
- Order processing
- Admin panel for product management

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Django (Python)
- Database: SQLite

## How to Run the Project
1. Install Python (3.10 or above)
2. Install Django:
pip install django

3. Run migrations:
python manage.py migrate

4. Create superuser:
python manage.py createsuperuser

5. Start the server:
python manage.py runserver

6. Open browser and go to:

http://127.0.0.1:8000/
## Output
Users can browse products, add items to cart, and place orders successfully.
